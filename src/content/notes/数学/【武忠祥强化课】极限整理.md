## 极限概念

### 1. 数列极限

- 极限定义
- 几何意义
- 数列极限**与前有限项无关**
- $\lim_{ n \to \infty }x_{n} = a \iff \lim_{ n \to \infty }x_{2k-1}=\lim_{ n \to \infty }x_{2k}=a$

### 2. 函数极限

**1) 当 $x \to +\infty$ 时：**

$$\lim_{x \to +\infty} f(x) = A \iff \forall \varepsilon > 0, \exists X > 0, \text{当 } x > X \text{ 时, } |f(x) - A| < \varepsilon.$$

**2) 当 $x \to -\infty$ 时：**

$$\lim_{x \to -\infty} f(x) = A \iff \forall \varepsilon > 0, \exists X > 0, \text{当 } x < -X \text{ 时, } |f(x) - A| < \varepsilon.$$

**3) 当 $x \to \infty$ 时：**

$$\lim_{x \to \infty} f(x) = A \iff \forall \varepsilon > 0, \exists X > 0, \text{当 } |x| > X \text{ 时, } |f(x) - A| < \varepsilon.$$
> [!abstract] 定理
> $$
> \lim_{x \to \infty} f(x) = A \iff \lim_{x \to +\infty} f(x) =  \lim_{x \to -\infty} f(x) = A
> $$

> 函数极限存在就是左极限和右极限存在且相等。

**4）当 $x\to x_{0}$ 时**

$$
\begin{array} \\

\lim_{x \to x_0} f(x) = A : \\
\forall \varepsilon > 0, \exists \delta(\varepsilon) > 0, \text{当 } 0 < |x - x_0| < \delta \text{ 时, } |f(x) - A| < \varepsilon.
\end{array}
$$


1. **$\lim_{x \to x_0} f(x)$ 与 $f(x_0)$ 无关**
    - 在定义中，$0 < |x - x_0|$ 意味着 $x \neq x_0$。
    - 极限探讨的是 $x$ 趋近于 $x_0$ 过程中的趋势，而不是在 $x_0$ 点本身的值。
2. **$\lim_{x \to x_0} f(x)$ 存在 $\iff f(x)$ 在 $x_0$ 在某去心邻域内有定义**
    - 即函数在 $x_0$ 的附近必须有意义（但不一定在 $x_0$ 点有意义）。
    - 图中手写补充了例子：$f(x) = \sin \frac{1}{x}, x_0 = 0$。

> 某点处极限存在只能说明该点的去心邻域内有定义，和该点处函数值**毫无关系**。并且，该点处极限存在是**去心**邻域有定义的充分必要条件，注意这里一定是去心❤。
